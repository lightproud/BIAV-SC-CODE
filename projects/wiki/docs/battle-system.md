# 战斗机制总览

> 忘却前夜的战斗是**指令卡 Roguelite**：每位唤醒体自带一套卡组，玩家用算力打出指令卡，靠界域体系、狂气爆发与命轮滚雪球。
> 本页为系统说明（机制层），具体到每位角色「怎么玩」见[玩法图鉴](/playstyle)。社区译名为准，数值随版本浮动。

[[toc]]

## 队伍与卡组

- 一支队伍编入数名唤醒体；每位唤醒体携带自己的**指令卡**与一张**灵知觉醒**卡。
- 每位唤醒体的卡牌大致分三类：

<p>
<span class="card-type-badge card-command">指令卡</span> 角色的常规招式，构成卡组主体，靠算力打出。<br>
<span class="card-type-badge card-spirit">灵知觉醒</span> 角色的觉醒/核心卡，往往是流派引擎（过牌、滚雪球、形态切换）。<br>
<span class="card-type-badge card-madness">狂气爆发</span> 狂气满后释放的大招，不耗算力。
</p>

::: tip 小学生比喻
指令卡 = 平时的普通攻击牌；灵知觉醒 = 角色的「必杀引擎」；狂气爆发 = 攒满怒气槽后免费放的大招。
:::

## 算力（Arithmetica）

- 每回合开局获得固定算力（基础 **5 点**），打指令卡消耗算力。
- 算力是每回合的「行动力预算」，部分角色/命轮能回算力或降卡费，从而一回合打出更多牌。

<p><span class="arithmetica-cost">⚙ 5</span> = 每回合初始算力</p>

::: tip 小学生比喻
算力 = 每回合的零花钱，每张牌标价多少就花多少，花光这回合就不能再出牌了。
:::

## 狂气与爆发

- 角色通过出牌、受击等积累**狂气**（上限 100）。
- 狂气满 100 即可释放**狂气爆发**（不耗算力的大招）。
- 多数角色可二次叠满触发**超限爆发**，把狂气上限从 100 提到 200，爆发更强。

<div class="madness-bar"></div>

<p><span class="card-type-badge card-madness">狂气 100</span> → 狂气爆发　|　再叠满 → <span class="card-type-badge card-madness">超限 200</span> → 超限爆发</p>

::: tip 小学生比喻
狂气槽 = 怒气条，满了放大招；超限 = 把怒气条上限撑大一倍，憋出更猛的大招。
:::

## 四界域

唤醒体按**界域**划分玩法体系，决定核心机制与配队方向。混沌可与任意界域混编。

| 界域 | 体系关键词 |
|------|-----------|
| <span class="realm-badge realm-chaos">混沌</span> | 可与任意界域混编 · 反击 / 打击 / 过牌 / 刷牌 |
| <span class="realm-badge realm-aequor">深海</span> | 触腕体系 · 深渊号令 · 触腕齐射 |
| <span class="realm-badge realm-caro">血肉</span> | 日服译「狂魔」· 胚胎 / 中毒 / 卖血 / 斩杀 |
| <span class="realm-badge realm-ultra">超维</span> | 超维空间 · 额外回合 · 斩杀 / 暴击 |

::: tip 小学生比喻
界域 = 角色的「门派」，决定他练什么武功路子；混沌是「百搭门派」，能跟谁都组队。
:::

## 命轮（神器）

- **命轮**是可装备的局外/局内强化遗物，给角色叠增益、改流派。
- 每位角色可装备数枚命轮（含高稀有度的核心命轮），叠位影响强度。
- 具名命轮常带叙事典故，资料见[道具故事](/item-stories)。

::: tip 小学生比喻
命轮 = 给角色戴的装备/法宝，戴对了能把他的特长放大好几倍。
:::
