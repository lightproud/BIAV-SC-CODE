**结论：没有。「命名权」不是作词者的法定权利，歌名归谁定，本质上是合同问题。**

- **著作权法里不存在「歌曲命名权」这一项权利。** 著作人身权的法定清单是署名权、发表权、修改权、保护作品完整性权——给作品起名不在其中。中国和日本的著作权体系在这点上一致。
- **歌名本身通常不受著作权保护。** 几个字的标题一般因过短、独创性不足而不构成作品，所以也谈不上谁「拥有」它。
- **实务惯例 ≠ 法定权利。** 歌名常常出自歌词，所以行业惯例上多由作词者**提出**名字；但**决定权**在歌曲的出资方 / 制作方（委托创作场景下尤其如此），除非合同明确写了「曲名由词作者确定」。
- **委托创作（游戏主题曲是典型场景）：** 权利归属以合同约定为准；合同没写命名事项，制作方定名、改名都不违法。唯一的边界是：如果改名同时篡改了歌词内容、或以损害作者声誉的方式使用，才可能触及修改权 / 保护作品完整性权。

一句人话翻译：作词者好比给房子设计了门牌样式的人，门牌上写什么字，最终还是房主（出资制作方）说了算——除非买房合同里专门写了「门牌归设计师定」。
