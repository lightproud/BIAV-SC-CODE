| 工具 | 功能 | git 依赖 |
|------|------|----------|
| memory_search | TF-IDF 语义搜索 + 4 维重排 | 无 |
| graph_query | 知识图谱 BFS 查询 | 无 |
| graph_related_files | 图谱关联文件 | 无 |
| store_facts | 事实存储 + 语义去重 | 无 |
| memory_utility | 文件效用 EMA 排名 | 无 |
| check_cache | 预计算缓存匹配 | 无 |
| recommend_context | 4 层融合上下文推荐 | 无 |
| rebuild_indexes | 全量重建索引 | 无 |
| character_persona | 角色人格生成 | 无 |
| **memory_writeback** | 变更检测 + 知识回写 | **git diff/log** |
| **session_briefing** | 会话 Briefing | **git log** |

BPT 方案：用 **文件 mtime 扫描** 替代 git diff/log，可选 SVN 集成。
