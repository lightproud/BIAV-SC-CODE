## 根因

当前的容错缺陷本质上是一个设计问题：**时间窗口是写死的常量，和实际运行间隔没有关系。**

```

├── CI 停 2h   → 没问题，窗口还覆盖得到

```

本次运行: 2026-04-12 08:00
→ 自动计算: lookback = 48h + 1h buffer = 49h
→ 无论停机多久，恢复后第一次就把缺口补上
