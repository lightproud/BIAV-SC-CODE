# 银芯指导 BPT 协议（草案 v0.1）

> 最后更新：2026-04-19 by 主控台（艾瑞卡会话）
>
> 状态：**骨架文档**，Phase 1.5 窗口内迭代完善。
>
> 触发决策：2026-04-19 战略转向——BPT 从银芯删除，银芯转为指导者；协议形态定为"人工对话搬运"。

---

## 一、协议目的

银芯不再在仓库内承载 BPT 代码，但 BPT 的开发仍需要银芯提供的能力：战略视角、架构经验、方法论、事实圣经、踩坑记录。

本协议定义**银芯 ↔ 守密人 ↔ BPT** 三方之间的信息流动规则，确保在没有自动化工具链的前提下，指导关系仍然可持续、可追溯、可沉淀。

## 二、角色与职责

| 角色 | 实体 | 职责 |
|------|------|------|
| **指导者** | 银芯主控台（长期战略锚点会话） | 输出设计意图、架构建议、方法论；不直接访问 BPT 仓库 |
| **搬运者 / 学习者** | 守密人 | 双向搬运对话 + 从指导中学习概念；是唯一跨越银芯/BPT 边界的实体 |
| **执行者** | BPT 会话（Claude Code harness） | 接收守密人转述的指令，在 BPT 仓库内执行 |
| **记忆层** | 银芯 `memory/` 档案 | 跨越会话的持久记忆，守密人可引用档案片段向 BPT 传达 |

## 三、信息流向（单向可追溯）

```
                     (Phase N-1 经验沉淀)
                            ↓
   银芯主控台 ──(概念/建议/档案引用)──> 守密人
                                          │
                                          │ (人工搬运)
                                          ↓
                                        BPT 会话
                                          │
                                          │ (成果 / 阻塞 / 追问)
                                          ↓
   守密人 ──(反馈 + BPT 状态摘要)──> 银芯主控台
                            ↓
                     (写入 memory/ 档案)
```

**关键约束**：银芯的指导只以"对话内容 + 档案引用"形式存在；银芯不向 BPT 推送代码、不订阅 BPT 仓库、不调用 BPT 的 API。

## 四、对话搬运的标准格式

### 4.1 银芯 → BPT 的指导包

每次守密人要从银芯向 BPT 搬运指导时，银芯应输出一个**可直接复制的搬运包**，包含以下字段：

- **指导主题**：一句话概括
- **背景上下文**：2-3 句话，BPT 会话可能缺乏的背景
- **具体建议**：方案 / 步骤 / 陷阱提示
- **引用档案**：银芯 `memory/` 中相关文件的路径与摘要片段
- **验收问题**：让守密人带回银芯的反馈问题（1-3 个）

### 4.2 BPT → 银芯 的反馈包

守密人从 BPT 会话带回的反馈应包含：

- **指导主题回执**（对应银芯上次的搬运包主题）
- **BPT 实际执行摘要**：做了什么、产出是什么
- **偏差或阻塞**：BPT 没有按原建议执行的地方、遇到的问题
- **守密人的学习记录**：从这一轮中理解到的概念（供银芯沉淀到 `lessons-learned.md`）

## 五、沉淀机制

本协议下产生的"银芯指导 BPT 经验"需在以下文件中积累：

- `memory/lessons-learned.md` — 搬运过程中的踩坑
- `memory/decisions.md` — 架构层面的联合决策
- `memory/bpt-guidance-log.md`（待创建） — 每轮搬运的主题 / 日期 / 成果概要（一行一条）
- 本文件 — 协议本身的迭代

## 六、边界（绝不做）

1. **银芯绝不直接访问 BPT 仓库**——即使技术上可行，也必须经守密人转述
2. **银芯绝不自动化指导流程**——守密人的"学习"是协议的核心价值，自动化会抽掉这个价值
3. **BPT 的内部决策不入银芯档案**——只有守密人公开认可并搬回银芯的经验才沉淀
4. **搬运包不含 BPT 的敏感代码片段**——银芯是公开层（BIAV-SC），不污染
