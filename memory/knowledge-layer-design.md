# 银芯知识层设计哲学 —— 神经符号白盒骨架（North Star）

> 守密人 2026-07-04 会话结晶（承 #386 运行时导航 / #396 全仓知识组织之后的定位对齐）。
> 本档是**知识层的北极星**：决定「什么该白盒化、边怎么长、怎么守边界」。工程改造以此为准绳。

## 一、一句话定位

> **OKF = 有结构的概念网络，承载白盒结构化知识；区别于神经网络承载的黑盒知识。**

它既不是「搜索」（在知识之内找某样东西），也不是「神经网络」（把知识化成不可读的分布式权重）。
它是**对知识的外延与形状的显式声明**——可枚举、可审计、可程序操作。

## 二、三个器官（不是三个对手）

| 器官 | 是什么 | 干别人干不了的 |
|------|--------|--------------|
| **骨架**：白盒概念网络（OKF/okf 层） | 结构 / 身份 / 层 / 边界 | 可枚举、可审计、可测试、可程序操作 |
| **结缔组织**：搜索 / 全文（ripgrep、kb_search 倒排） | 钻进骨架「只指不放」的血肉 | 模糊、无界、按内容命中 |
| **肌肉**：黑盒神经（艾瑞卡本体 LLM） | 推理 / 泛化 / 生成 | 处理没见过的、插值（有脑补代价） |

艾瑞卡是肌肉（会脑补），OKF 是骨架（不脑补、但可能漏）。二者互为解药：肌肉给骨架泛化力，
骨架给肌肉「你到底知道什么、到没到边」。

## 三、白盒 vs 黑盒的互补对立

- **黑盒（神经）**：会泛化 / 插值 / 处理未见 / 生成；但不可审计、不可枚举、会脑补、无稳定边界。
- **白盒（概念网络）**：精确 / 可枚举 / 身份稳定 / 有界；但不会泛化、只知道被显式建模进去的。

**招牌死法各异**：黑盒**以捏造撒谎**（confabulate）；白盒**以遗漏撒谎**（假完备——有界地图感觉是全的，
其实只和策展一样全）。「可枚举边界」这同一性质，既是白盒最强的**幻觉护栏**，又是它最深的**假完备陷阱**。

## 四、由定位导出的三条工程命令

1. **白盒只花在骨架上**（放指针不放本体的更深形态 = 白/黑盒边界裁定）：
   结构进白盒（哪些平台、哪层、怎么连），血肉留黑盒/搜索（750 万条正文、44M 解包本体原地）。
   **判据**：一个概念若没有非平凡的边、也没有搜索给不出的层/身份元数据，它就是「被冻进文件的搜索结果」——
   该去挣一条高信号边，或退回给 ripgrep。扁平孤立指针 = 付了白盒建模的代价、没买到白盒的好处。
2. **把不变量测起来**（白盒 = 唯一能写测试、能被程序操作的知识）：
   传递闭包 / 集合运算 / 一致性 / 完备性——这些对黑盒做不到。银芯已有的 data_layer 纪律、指针不落空、
   覆盖哨兵**能存在，纯因知识是白盒的**。可审计不是附赠性质，是让知识**可治理**的前提。
3. **死守覆盖哨兵**（守白盒的招牌死法）：假完备是白盒唯一的特征危险，覆盖哨兵是它唯一的守卫——
   边界必须是**真边界**，不是伪全集。

## 五、神经系统的两条设计教益（拓扑像脑，但要守纪律）

- **符号底座 + 神经式动态**：底座保持符号（可审计），只让**检索动态**变神经——
  **扩散激活（spreading activation）**：从命中概念沿带类型的边多跳扩散、带衰减，返回被点亮的子图 = 联想召回。
  这是平铺搜索与单跳查都给不了的第三种检索原语，也是「概念网络 ≠ 搜索」的杀手级消费方式。
- **像脑地剪枝**：神经系统的智能来自**突触修剪**（留高信号、砍噪声），不来自堆突触。
  #393 删画师边（72 角色同画师 = 零区分度噪声星）字面就是一次剪枝。
  **维度的价值不在多，在「每条边是否携带别处得不到的信号」。**

## 八、Pillar A 的教益：诚实分类 > 强行连接

A 原设想「给孤立层派边压孤立率」。实测证明：在「不造噪声星」（守密人选 1）约束下，
**真数据 derive 的高信号边就那么多，剩余孤立不是缺陷、是参考层的本相**。于是 A 从「加边」
转为「**显式声明两层结构**」——让 skeleton（真网络）与 search（参考层）成为白盒对自身形状的
可断言声明。这本身就是命令一「白盒只花在骨架上」的兑现：**不把血肉硬塞进骨架，而是诚实标出
哪是骨架、哪是靠搜索够到的血肉**。度量的价值不在「孤立率降到 0」，在「每个孤立都是有意的」。
