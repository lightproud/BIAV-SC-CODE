§3 对各开发者职能的建议

每条锚定 06-02 证据，「建议」属银芯输出层判断。

制作人 / 运营总控（Producer）

两件事当日最值得接住：① 营收/在线破纪录（俄语区观察）——周年验证了「我们一起救活了它」的叙事，建议官方用一次数据向的真诚回顾把热度转成长期信任；② 「取消 300 抽加三围」抗议——公告频道被同一诉求刷屏（要求新三围系统给 50%+ 加成），是组织化民意，沉默会发酵，建议明确回应该系统设计意图或调整空间。A

数值 / 平衡策划（Balance）

三条平衡侧主线：① 沙耶 0 费弃牌争议——跨中/越/俄语区共识「放完腐蚀后弃牌会丢掉你需要的牌」「带沙耶胚胎积累慢一倍」，联动主角的核心机制存在体感摩擦，建议复盘 0 费牌的弃置逻辑；② 2.4.0 难度膨胀——俄语长评「怪物血量涨了 3-5 倍，连活动都要靠复活，从享受变成啃怪」，是明确留存风险；③ 日服/国际服印记数值差——韩区指「日服 1% 印记 = 1% 腐蚀（0 费），国际服只有一半 0.5%」，区域平衡差异已被察觉，建议给出口径。B

主文案 / 叙事（Narrative）

叙事投入度在 06-02 体现为玩家自发的 Arc3 路线推演（GTulu+三周年收束 Arc2→3.0、翻出 Light 的领域精通重做承诺）与情感共鸣（Reddit「What a moving and beautiful scene, it really hits home」）。建议：在合适节点给 Arc2 收束 + Arc3 框架一个弹性预告，承接这份考据热情。B

本地化（Localization）

新人侧出现默认语种/服务器困惑（Reddit「Steam 入坑，默认语言不是英文，我在国际服吗」）。建议：Steam 首次启动加一步明确的语种/服务器确认。正面信号：西语区盛赞经济慷慨（「每天送 500 碎屑」「yousavedmorimens 再给 100 抽」），可作拉新话术延续。B

§5 Bug 详情（重现步骤 / 预期结果 / 实际结果）

Bug 1 — 塔薇 + 沙耶「羽种」显示与消耗 B

  • 来源：official-q-a 中文玩家两段式反馈。
  • 重现步骤：① 队伍同时上塔薇与沙耶；② 用塔薇打出沙耶的 2 费牌生成羽种；③ 观察状态栏羽种数量；④ 用沙耶大招尝试消耗羽种。
  • 预期结果：两角色羽种「各 1 个」或「共 2 个」正确显示；羽种作为「可被沙耶大招消耗的资源」应能被沙耶消耗。
  • 实际结果：状态栏只显示「1 个」羽种；塔薇产出的羽种无法被沙耶大招消耗。
  • 影响：直接影响沙耶联动队的核心资源循环，建议优先复现。

Bug 2 — Steam 新装登录崩溃 B

  • 来源：official-q-a 英文新玩家，「装好 30 分钟，点 Enter Mythag 黑屏一下就退，啥也没发生，用了 fix 仍旧」。官方回应追问是否换 Facebook/Steam 账号登录、手机端能否打开，玩家已生成 bug report。
  • 重现步骤：① Steam 全新安装；② 启动点「Enter Mythag」；③ 观察是否进入。
  • 预期结果：正常进入登录/主界面。
  • 实际结果：闪现加载界面后退出，停在原地无法进入。
  • 影响：发生在 PV 拉新高流量期，直接折损新玩家首日留存。

Bug 3 — 客户端自动登出（UI 仍显示已登录） B

  • 来源：越南语玩家，「这两天领 Steam 免费版有问题……App 自己登出了，但 UI 看起来还是登录状态」，已提 support ticket；曾误判为越南网络问题，挂 VPN 后无异常。
  • 重现步骤：① Steam 免费版运行一段时间；② 观察登录态与 UI 显示是否一致。
  • 预期结果：登录态与 UI 一致；掉线应有明确提示。
  • 实际结果：后台已登出但 UI 仍显示已登录，操作异常。
  • 影响：误导玩家、打断体验，疑与区域网络/会话保活有关。
