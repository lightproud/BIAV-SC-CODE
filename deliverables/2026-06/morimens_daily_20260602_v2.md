## §5 Bug 详情（重现步骤 / 预期结果 / 实际结果）

### Bug 1 — 塔薇 + 沙耶「羽种」显示与消耗 <span class="badge badge-b">B</span>

- **来源**：official-q-a 中文玩家两段式反馈。
- **重现步骤**：① 队伍同时上塔薇与沙耶；② 用塔薇打出沙耶的 2 费牌生成羽种；③ 观察状态栏羽种数量；④ 用沙耶大招尝试消耗羽种。
- **预期结果**：两角色羽种「各 1 个」或「共 2 个」正确显示；羽种作为「可被沙耶大招消耗的资源」应能被沙耶消耗。
- **实际结果**：状态栏只显示「1 个」羽种；塔薇产出的羽种无法被沙耶大招消耗。
- **影响**：直接影响沙耶联动队的核心资源循环，建议优先复现。

### Bug 2 — Steam 新装登录崩溃 <span class="badge badge-b">B</span>

- **来源**：official-q-a 英文新玩家，「装好 30 分钟，点 Enter Mythag 黑屏一下就退，啥也没发生，用了 fix 仍旧」。官方回应追问是否换 Facebook/Steam 账号登录、手机端能否打开，玩家已生成 bug report。
- **重现步骤**：① Steam 全新安装；② 启动点「Enter Mythag」；③ 观察是否进入。
- **预期结果**：正常进入登录/主界面。
- **实际结果**：闪现加载界面后退出，停在原地无法进入。
- **影响**：发生在 PV 拉新高流量期，直接折损新玩家首日留存。

### Bug 3 — 客户端自动登出（UI 仍显示已登录） <span class="badge badge-b">B</span>

- **来源**：越南语玩家，「这两天领 Steam 免费版有问题……App 自己登出了，但 UI 看起来还是登录状态」，已提 support ticket；曾误判为越南网络问题，挂 VPN 后无异常。
- **重现步骤**：① Steam 免费版运行一段时间；② 观察登录态与 UI 显示是否一致。
- **预期结果**：登录态与 UI 一致；掉线应有明确提示。
- **实际结果**：后台已登出但 UI 仍显示已登录，操作异常。
- **影响**：误导玩家、打断体验，疑与区域网络/会话保活有关。
