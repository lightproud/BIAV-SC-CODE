## 〇、目标定位（拷问 #1 裁定）

**门禁按甲、覆盖按乙**：验收线与权重服务 **BPT Desktop 换装信心**（关键路径严格、
非关键允许已知差异清单）；测试**内容全量铺**（146 行审计矩阵全建全跑全记分），
非关键红灯只进记分牌不拦门。全量记分牌随时可抽出当追平证据，但不绑架验收。

**结构性天花板承认在先**：官方系统提示词专有（净室禁复制），逐比特轨迹一致原理上
不可达（POSITIONING §2/§4）。目标三级降格：协议层完全一致（可 100% 断言）→
结果层任务终态等价（统计带）→ 决策层行为相似度（只记分不判死）。

## 一、五层金字塔

| 层 | 测什么 | 判定 | 跑在哪 | 模型 |
|---|------|------|------|------|
| L1 流语法 | SDKMessage 序列形状/字段类型/变体判别 | 归一化精确 diff | 无钥 CI 常跑 | 内容盲仿真器 |
| L2 选项语义 | 每个 Options 字段可观测效果（146 行矩阵派生决策表） | 决策表断言 | 无钥 CI | 仿真器 |
| L3 工具差分 | 同输入→同工具输出（两臂活体驱动） | 归一化对比 | 无钥 CI | 仿真器（脚本 tool_use 驱动） |
| L4 故障注入 | 429/5xx/断流/abort 恢复语义 | 状态机断言 | 无钥 CI | 仿真器 |
| L5 端到端等价 | 同任务→终态/通过率/轮数 | 统计带（乙门禁） | dispatch 花预算 | 真模型（haiku） |

**心脏 = 内容盲差分仿真器**（spike 已验活体可行）：两引擎接同一台本地
Messages-API 仿真器，模型侧确定性同脚本，流差异即引擎真差异。内容盲纪律
按决策条执行（请求体不读不存 + 自审断言）。

**双包同钉**（spike 意外收获推论）：官方 agent-sdk 版本与其 spawn 的 claude-code
引擎版本**各自锁死**（pins 档案），否则基线脚下滑动（实测 CLI 2.1.201 已发
`active_goal` 等 0.3.199 类型面之外的变体）。官方包一律 `npm i --no-save`
运行时装、绝不进仓库依赖。

## 二、L5 判定标准（拷问 #4 裁定：乙）

- **门禁唯一一条**：全任务集聚合通过率不低于官方，容忍差 ≤5%；单任务允许互有胜负
- **轮数/成本/墙钟只记分不拦门**（进趋势曲线；实证依据：run #74 墙钟 2.5× 差异
  与 token 量不符，纯 API 延迟噪声——拿噪声轴当门禁必翻车）
- 参数：**repeat=5 取中位数**、模型 haiku、**每轮预算帽 ~$1.5**、
  节奏 = 手动 dispatch + 官方基线升版必跑

## 三、L5 任务库（拷问 #5 裁定，Desktop 剖面守密人亲述回流校准）

五大维、中文贯穿变体轴、合计 15-18 任务：
**聊天问答**（纯对话零工具）/ **知识检索**（跨文件找与汇总）/ **文档处理**
（长文读改写）/ **代码工程**（单改·多改·自验证·反硬编码·工具混编五子维，
含硬任务 10/11）/ **长会话**（多轮流式）。

- **拒绝边界类不进门禁**（专有提示词影子，最多记分观察）
- **图片生成判为射程外**：Desktop 的他模型生图走 host 层/MCP 层，官方 agent-sdk
  引擎无可对照行为；作「provider 可替换性 = 定位硬承诺」注脚，本 SDK 只需不挡路

## 四、L6 行为指纹（拷问 #6 裁定：后置 + 零成本留痕）

L5 官方臂每次运行**顺手留存公开消息流 JSON**（合法观测面、零边际成本）。
L6 指纹提取器（步序模式/并行度/首工具分布→相似度分）等两个触发条件之一再开工：
① 提示词迭代线重启需要靶子；② L5 出现「正确性平但体感不对」需解释工具。
只记分，永不设 CI 门禁（天花板所在）。

## 五、明知不测什么

输出文本风格/措辞/排版；拒绝边界逐例一致；官方 CLI 内部件（settings 引擎全量/
沙箱/OTel——审计已裁 N/A-by-design）。
