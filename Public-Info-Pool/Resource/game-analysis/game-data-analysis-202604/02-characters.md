## 2.5 角色设计哲学（事实圣经）

来自 interview-2026-04.json 和 design-decisions.json 的制作人一手陈述：

**最早设计的角色**：塔薇（Tawil）和朵尔（Doll），立项初期即创作。

**"创世纪角色"**：环行·拉蒙娜、熔毁·朵尔、血链·希洛、诞妄·墨菲四个变体同期构思。此外还有一个"超维界域创世纪角色"概念已定稿但未公开。

**制作人最喜欢的角色**：环行·拉蒙娜——技能组由制作人 Light 亲手设计，"复刻上一张打出的卡"体现"从过往经历中汲取力量"的主题。V1.1 PV 结尾由 Light 亲自加入台词："无论失败多少次，我最终一定会拯救你。"

**"24"的设计例外**：将四个角色生态位压缩到一个单元，不划算但为"人格分裂"叙事服务。战斗设计师们"为理想燃烧了一次"。

**萝坦的设计理念**：第一个打击流角色。"不是因为游戏需要打击流派才设计，而是因为萝坦的能力与人设适合用打击的手感与循环去表达"——体现"玩法与叙事统一"原则。

**设计三原则**（制作人原话）：
1. 玩法与叙事统一——卡牌表达角色身份，而非单纯数值
2. 每个选择都有意义——每种队伍配置、遗物、出牌顺序都应创造独特策略权衡
3. 只增强不削弱（尽可能）——抽卡是服务承诺；不得不削弱时提供退还机制
