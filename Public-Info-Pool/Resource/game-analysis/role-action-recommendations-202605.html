§0 三条跨职能最高优先级

在分职能细化之前，三件事跨职能、最高优先、且证据最硬：

  1. 活动难度与奖励解耦（制作人 + 战斗 + 系统）：周年「全服挑战」难度是本窗口唯一系统性 负面，官方已紧急回退（敌人生命 -35%、伤害 -15%），应固化为常态设计纪律而非一次性救火。
  2. 守护「白嫖必然性」信任契约（制作人 + 系统 + 运营）：「长期必然集齐」是 F2P 口碑根基， 任何削弱必然性的改动都动摇信任。
  3. 机翻本地化人工润色（文案 + QA + 制作人）：西/俄/泰/越/葡五语区共性短板，低天花板、 高频次，持续侵蚀新玩家第一印象，尤其在东南亚/葡语高黏腹地。

§1 制作人 Producer

  1. 把「奖励解耦高难门槛」立为大型活动设计红线。 依据：周年全服挑战难度是窗口唯一 四源交叉印证的系统性负面（Steam 负评首因 / YouTube「最掠夺贪婪的周年活动」/ 中文 Discord 「新手哪来 +12 80 级」/ 韩语肝度）；官方已 -35% 生命 /-15% 伤害回退（已落地），证明可修复， 应升级为纪律。
  2. 守护「最终你会拿到所有人」的白嫖信任。 依据：英文 team-building「you are realistically getting ALL of them」、葡语盘点「只差 4 个就集齐」——这是慷慨 F2P 口碑的 机制根基。
  3. 把「被低估神作 / 起死回生」立为官方品牌主叙事。 依据：跨平台最强正向资产——YouTube 窗口内新片「How a Dying Game achieved Immortality」（4 天前 4.95 万）+ Steam「should be WAAAYYY bigger」+ 制作人公开信「你们曾拯救了忘却前夜」三者同构。
  4. 联动选品延续「调性契合优先」。 依据：沙耶（Nitroplus 克苏鲁）与世界观高契合、拉新 精准；但越南语预警「锣鼓太响成大杂烩，日不日中不中」，调性稀释是真实风险。
  5. 评估「永久授权」模式的供给可持续性。 依据：泰语玩家洞察「永久 license 条件难找大 IP 合作」（推断但具产业逻辑）；玩家友好 vs IP 供给的权衡需进长期规划。
  6. 节制「共渡难关」情感叙事的频率。 依据：制作人公开信中文优先发布精准触达核心盘，但 部分玩家反向解读「是否真缺钱」（推断）；情感契约须与内容兑现同步，否则被读为焦虑信号。
  7. 建立跨窗口连续监测，而非单窗口快照。 依据：本窗口情绪势能大量面向窗口外（沙耶卡池 05-30、白嫖 06-06）；真实成败需 06 月上旬按六锚点补做闭环复盘。

§2 增长运营 Growth Ops

  1. 把 YouTube 发现性流量接到新手留存路径。 依据：「起死回生」窗口新片带来漏斗顶端 增量，但英文 game-question（691 作者、人均强度最低）显示大量新玩家「问完即走」，转化在 新手段渗漏。
  2. 放大「克苏鲁 + Slay the Spire + 慷慨 F2P + 沙耶白嫖」组合钩子。 依据：Steam 正评 反复出现的母题；「originally got it for saya but came out really enjoying it」。
  3. 以沙耶卡池（05-30）+ 白嫖（06-06）做拉新承接波。 依据：250 抽免费沙耶是稀缺善意 钩子；ruliweb 韩国因联动爆发「척추 서는 콜라보」发现潮。
  4. 定向投放东南亚 / 葡语腹地。 依据：印尼 15,253 + 越南 11,942 + 葡语 10,196 高黏， 人均发言强度前三，被现有 EN/CN 重心低估。
  5. 与 KOL 协作而非自建攻略，新角色上线前供准确机制信息。 依据：Flares 英文评级被 Discord 直接引为决策依据，璐洛洛（38.6 万）/恒燐主导中文；防错误评级先入为主。
  6. 兑换码 / 周年货币做引流闭环。 依据：上线日兑换码被高频转发；周年银芯兑换「换五万多」 获社区高共鸣。
  7. 量化「发现→安装→留存」转化漏斗。 依据：多条「为沙耶而来、留下来享受游戏」转化记录 待数据化追踪。

§3 社区运营 Community Ops

  1. 优先保障 game-question 答疑健康度（置顶 FAQ / 自助化）。 依据：691 作者最广覆盖， 是新玩家主沉淀地，直接决定早期留存体验。
  2. Reddit 轻介入：置顶权威攻略 + 澄清机制误解。 依据：低冲突（抱怨仅约 2%）、高攻略 密度，最适合官方介入且不被实时噪声淹没。
  3. 维护东南亚 / 葡语核心成员关系。 依据：葡语 44 人、越南 61 人撑起万级消息量，关键 成员流失即塌方，是高价值高脆弱群体。
  4. 以透明沟通对冲技术瑕疵。 依据：Arachne 生日邮件故障的主动说明获 ❤18 高共鸣；坦诚 沟通能有效化解负面。
  5. 监测但不放大社区纷争。 依据：「隔壁孝子煎炒」口水摩擦属生态噪声，无实质影响；护 制作人阵营「不喜欢可以不玩」自带舆论免疫力。
  6. 缓解核心广场「硬核浓度过高」。 依据：「我已经得 300 抽恐惧症了」——普通玩家被 meta/ 抽卡讨论挤压，可设休闲向频道分流。
  7. 扶持二创生态。 依据：微博「画」444 次、Reddit 图片帖占三成、CP32 线下摊位、Goliath 生贺；二创是黏性核心来源。
  8. 收编各语区自建信息站。 依据：韩国 morimenz-kr.github.io 自建、印尼自发分发活动 入口；可官方背书 / 导流，省运营成本。

§4 战斗策划 Combat Design

  1. 大型活动难度分档，奖励解耦高难。 依据：全服挑战劝退新手；官方已回退（已落地）， 应作为设计纪律延续。
  2. 修复数值断层。 依据：Steam 长评「借来的角色 0 费无限复制打 60k，我的角色打 6 点」； 养成深度与体感断层并存。
  3. 平衡命轮溢出风险。 依据：玩家精算「命轮重复 / 抽出无角色命轮时溢出」、「+12 前该 停手以防再抽到」——机制存在浪费风险。
  4. 维持「同色协同 vs 跨色泛用」的命域张力。 依据：命域共鸣（如蓝域开局额外触手）是 配队深度根源与玩法识别度。
  5. 关注顶级辅助过度集中（Thais）。 依据：「basically requires Thais for function」； 单一辅助强制化会压缩构筑多样性。
  6. 联动角色需配套体系支撑。 依据：泰语「沙耶加速 Exalt，但队里无 Exalt 输出则收益 有限」；沙耶需 Exalt 生态。
  7. 延续共斗（D-Tide）机制改良。 依据：德语好评「不再被肝帝开放一小时内 farm 光」， 本期优于往期。
  8. 输出位 tier 生态与官方意图对齐。 依据：Flares 评级（Vortice A-/Mouchette C 等）已成 社区共识，官方需以数据对话防分歧。

§5 文案策划 Narrative / Writing

  1. 把克苏鲁叙事编入日常玩法循环。 依据：叙事是最强拉新卖点（YouTube/Steam），但 lore 频道日常体量薄（英 1,902 / 中 76）——卖点尚未转化为日活驱动力。
  2. 把控沙耶剧情（06-06）与既有世界观的衔接质量。 依据：lore 频道升温提前考据沙耶 衔接；越南语「大杂烩」调性风险需在剧情放出后重点监测。
  3. 修复翻译不一致。 依据：英文 lore「now there's two conflicting translations, which one is real biav」——同一内容多版译文损害考据信任。
  4. 机翻本地化人工润色（剧情关键段 + UI 高频词优先）。 依据：西（ChatGPT）/俄（神经网络 翻译）/泰（性别称谓乱）/越（lỏ）跨语区共性短板。
  5. 提供中文原文 / 术语对照给海外考据党。 依据：「wish we had the original chinese instead of only the edgy coolsounding english」。
  6. 保留「点到为止暗示宏大」的叙事魅力，但补关键展开。 依据：「morimens lore hints at huge stuff that I desperately need elaborated on」——魅力与痛点一体两面。
  7. 深化角色心理档（psyche deepdive）。 依据：Kath 心理档「good story, I like kath a lot」获正向，是留存考据党的高价值内容。

§6 系统策划 Systems Design

  1. 为高难内容（融灾）提供低强度兜底获取路径。 依据：融灾自动移除 + 90 级材料路径 改动 →「劝退我这种付费轻玩家」；该群体 LTV 最健康。
  2. 全服累计目标设可达档位。 依据：「300 多把也凑不满 1000000」百万目标不可达。
  3. 谨慎改动养成材料路径。 依据：「90 材料只剩融灾，为何改动」——改动直接触动留存神经。
  4. 改动前评估影响面与沟通（如 250 空幻）。 依据：「害到的只有囤抽课佬 / 特定角色厨， 正常人用得完」——影响有限但需提前说明。
  5. 降低命轮经济的认知门槛。 依据：命轮复杂度 + 溢出风险是新玩家进阶门槛。
  6. 缓解内容肝度（homework fatigue）。 依据：韩语「每加一个玩法，光作业量就追得吃力」； 肝度逼近临界。
  7. 放宽新手助战（主线首通后可借）。 依据：韩语建议「首通后能借，对新手更亲切」。
  8. 评估慷慨发放对满命平衡的长期影响。 依据：韩语「一次发 500，满命基准平衡恐保不住」 ——延迟显现的隐忧。

§7 客户端 Client

  1. 修复加载 / 卡顿 / 中途无响应。 依据：Steam「the game doesn't load」、俄「вообще не работает」、App Store「always not responding halfway」、「卡顿明显」。
  2. 优化 UI 层级与信息密度。 依据：Steam 长评「50 different menus、35 ongoing events、 每 5 分钟任务后闪烁奖励」——菜单臃肿。
  3. 保持多端性能优化口碑。 依据：「Linux/MacOS day 1, very optimized」获赞，但移动端 有卡顿；优势需守住。
  4. 补齐局外展示功能（炫彩等）。 依据：中文「花种没法局外看炫彩，必须进去打才能看」。
  5. 优化补丁下载 / 客户端体积。 依据：「network server too busy to process my patch download」导致卸载。

§8 服务端 Server

  1. 优先排查俄区 VPN 登录问题。 依据：俄语反复「必须开 VPN 否则进不去，以前不用也行」 ——疑似近期变化（推断），直接劝退俄语区。
  2. 核查账号绑定 / 找回链路。 依据：Google Play「很多账号绑了邮箱直接没了」——若属实 为高危数据事故。
  3. 保障版本维护 / 开服稳定性。 依据：上线维护是脉冲峰值节点，稳定性直接影响首日体验。
  4. 根治邮件 / 发奖系统可靠性。 依据：Arachne 生日邮件因代码故障延迟；虽透明沟通化解， 仍应根治。
  5. 主动监控区服可达性（分区延迟 / 封锁）。 依据：俄区可达性退化；需主动监控而非被动 等玩家反馈。

§9 GX 游戏体验 / UX

  1. 新手引导自助化，降低 game-question 答疑负载。 依据：691 作者「问完即走」，转化漏斗 在新手段渗漏。
  2. 简化首屏与菜单信息架构。 依据：「50 menus、flashing rewards every 5 min mission」 ——信息过载劝退。
  3. 平滑养成曲线，缓解肝度感。 依据：韩语「작업량 追不动」——体验疲劳。
  4. 借助战体验前置（主线首通即可借）。 依据：韩语建议降低新手上手门槛。
  5. 引导玩家理解命轮 / 命域机制。 依据：命轮溢出 / 同色协同对新玩家不透明，是认知门槛。
  6. 打磨「上手首小时」体验。 依据：西语「tutorial battle 不该难，该引导却没引导」—— 首战体验劝退。
  7. 减少跨期活动叠加的认知负担。 依据：「35 ongoing events」并发过载。

§10 QA

  1. 复现并验证「常驻天赋强化 5 级跨角色自动适用」。 依据：韩语「强化一个常驻角色，限定 角色全自动适用，是 bug 吗」——疑似机制缺陷。
  2. 回归测试账号绑定 / 丢失。 依据：「账号绑邮箱直接没了」——数据安全高危，须复现闭环。
  3. 验证借用角色数值异常。 依据：「borrowed char 0 费无限复制打 60k」——平衡 / 利用 漏洞。
  4. 客户端加载与多端兼容性矩阵测试。 依据：多语区 load / 卡顿 / 无响应反馈。
  5. 本地化 LQA（性别称谓 / 文本错误）。 依据：泰语性别称谓混乱、西语「act 2 文字错误 更严重」。
  6. 区服登录 / VPN 场景化测试。 依据：俄区可达性退化，需场景复现。
  7. 具名缺陷跟踪闭环（如 Soulforge bug）。 依据：Reddit「Soulforge bug」帖。

§11 美术 Art

  1. 强化「水彩厚涂克苏鲁质感」标志风格。 依据：Steam 高频赞「流动、粘稠、非人的克苏鲁 质感」「DARKEST gacha storywise」——差异化品牌资产。
  2. 提升立绘与战斗小人的一致性。 依据：Arachne「立绘下半身是蜘蛛，与战斗小人对不上」， 玩家 / 绘师困扰；一致性提升沉浸与二创可还原度。
  3. 评估增加男性角色。 依据：多条「we def need more male characters」、中文「你游兄妹 总有一方没立绘」。
  4. 管理「魅男」属性与福利尺度的调性。 依据：魅男争议（Steam 激烈差评 + 微博「人设很 那个」）与英文区对 Thais 主界面尺度的调侃并存；受众分化需平衡，属长期调性议题。
  5. 官方支持角色生贺 / CP 二创。 依据：Goliath 0520 生贺、设计师解禁稿、CP32 线下—— 二创号召力强。
  6. 维护角色昵称的本地化生态。 依据：中文「鸟哥鸟弟 / 羊妈 / 淡墨」、德语「Erica Enjoyer™」——角色已成社区共同语言。
  7. 延续 ED / 概念 PV 的音画产出。 依据：ED「来自遗忘」MV 具独立传播力，概念 PV 系列 持续产出，是低成本高传播的内容资产。

§12 信度与使用说明

  • A 级（硬事实）：Steam 86% 好评、各频道消息量与作者数、官方难度回退补丁、官方公告 时间线——可直接据社区归档复算。
  • B 级（多源强归纳）：周年难度系统性负面、本地化共性短板、东南亚/葡语高黏高脆弱、 Thais 版本核心——多平台交叉印证。
  • C 级（推断，已标注）：永久授权供给约束、制作人信「缺钱」解读、俄区 VPN 系近期变化 ——供参考不作定论。

各职能可据此分配采信力度：A 级可直接立项，B 级宜小步验证，C 级仅作监测线索。沙耶联动核心 兑现（05-30 / 06-06）在本窗口之后，建议 06 月上旬据新数据复核本建议书。
