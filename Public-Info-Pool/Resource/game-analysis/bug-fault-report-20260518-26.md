## A 模块 · 客户端稳定性与性能

本窗口最高频的技术故障类别。崩溃 / 卡死 / 黑屏 / 加载失败集中在移动端，多条已随
V2.5.1.1 维护修复，部分加载与发热问题仍有零星反馈。

### A-1 关卡内卡死 + 疯狂闪退 `官方已修复`

- **重现步骤**：iPhone 13 / iOS 26.4.2，进入关卡战斗，连续作战一段时间后触发。
- **实际结果**：画面卡死——可点退出、点物品、点角色，但角色无法在格子间移动；点「修复」按钮无效，需强退重进。
- **预期结果**：关卡战斗全程可正常操作格子移动，不发生卡死。
- **来源 / 状态**：official-q-a 玩家报告（含设备信息）→ 标「已解决」（05-19）。

### A-2 高画质高帧率发热闪退 `官方已回应`

- **重现步骤**：iOS 设备开启高画质 + 高帧率，进入超维(Ultra Realm)、Boss 本，或释放高特效大招（如环行拉蒙娜、克莱门汀），或通关后观看剧情。
- **实际结果**：设备过热并闪退。
- **预期结果**：长时间高负载场景不因发热崩溃。
- **来源 / 状态**：official-q-a → 官方回应「设备过热所致，建议在设置中调低画质/帧率改善」（05-19，硬件层规避，非代码修复）。

### A-3 启动后 Logo 消失即黑屏 `官方已修复`
